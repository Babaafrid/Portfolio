# Portfolio (Vite)

This project was migrated from Create React App to Vite for a faster development experience.

Available scripts:

- `npm run dev` — start Vite dev server (default port 3000)
- `npm run build` — build for production (output to `build/`)
- `npm run preview` — preview the production build locally

Notes:
- The entry file for Vite is `index.html` at the project root which loads `/src/main.jsx`.
- CRA `public` directory is preserved for static assets, but Vite serves the root `index.html`.
